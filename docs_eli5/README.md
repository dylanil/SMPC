# docs_eli5 — plain-English versions of the project's documents

This folder holds plain-language ("explain it like I'm a busy product manager") companions to
the project's main documents. They're written for a reader who needs to understand the *concepts
and decisions* without every technical nuance.

**Two ground rules for everything in here:**

- The **originals are untouched and remain the source of truth.** These are companions, not
  replacements. If an `_eli5` file and its original ever disagree, trust the original.
- Every `_eli5` file keeps the **same finding codes** (RB-01, M1, G-H1, …) as its original, so you
  can always jump to the original for the full technical detail.
